## Description
XReplyGPT is a simple yet powerful Chrome extension that automatically generates replies to tweets on Twitter/X. It uses **Google Gemini AI** to analyze tweets and craft appropriate and relevant responses. This extension is designed to save you time and effort in engaging with your followers by providing quick and personalized replies.

## How it works?
1. Get a free Google Gemini API key [here](https://makersuite.google.com/app/apikey). No credit card required!
2. Save it in the extension's config section.
3. Open https://x.com.
4. Press the shortcut CTRL+SHIFT+L (or CMD+SHIFT+L on Mac).
5. Click on the generated reply to send it.

### Available Models
- **Gemini 2.0 Flash (Experimental)** - Latest model with best performance (recommended)
- **Gemini 1.5 Flash** - Fast and efficient, stable release
- **Gemini 1.5 Flash-8B** - Lightweight model for quick responses
- **Gemini 1.5 Pro** - Advanced reasoning capabilities
- **Gemini Pro** - Standard model
